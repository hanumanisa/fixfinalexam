Objective

Student Data Analytics for Improving Academic Experience and Success. This project aims to assist lecturers and academic institutions in improving students' learning experience and academic success through analyzing historical lecture data. By utilizing data science methods, this project consists of four main use cases that are interrelated.

User Target

  • Lecturers
  • Academic Administration

Quick Facts

  • 4 Use Cases
  • 3+ Data Science Methods
  • 2025 Project Year
  • 100% Data-Driven

Use Cases

1. Semester Recommendations for Courses

This project aims to recommend the most suitable semester to take a specific course based on historical academic data. By analyzing patterns from previous student performances, the system can:

  • Predict expected grades for each available semester
  • Help students plan their academic journey strategically
  • Improve overall academic performance by suggesting optimal timing

With this system, institutions can be faster in taking action that suits the needs of each student, not only based on final grades, but also based on learning patterns and their activeness in class.

2. Course Difficulty & Enrollment Analysis

This project aims to predict the difficulty levels of courses within the same semester by analyzing historical enrollment data, student grades, and course difficulty indicators. Through this system, academic planners can:

  • Predict the relative difficulty level of each course
  • Recommend optimal course combinations within a semester
  • Assist students in making informed enrollment decisions
  • Provide data-driven insights for curriculum teams

By considering not only grades but also enrollment patterns and course difficulty history, institutions can create academic plans that better support student achievement.

3. Instructor Performance Clustering

This project aims to group instructors based on student grade, attendance percentage, semester, total students, and difficulty level of the course. By doing so, we can:

  • Identify high-performing instructors
  • Detect teaching patterns affecting student outcomes
  • Provide insights for evaluation and improvement

The clustering is done using unsupervised learning to reveal natural groupings without predefined labels.

4. Student Learning Style

The purpose of this modeling is to predict a student's learning style based on two main data points: average score and attendance percentage. By knowing the prediction of student learning styles, we can:

  • Identify students who excel and are diligent
  • Find students who are smart but rarely attend
  • Recognize students who are diligent but have low grades
  • Identify students who have problems with both attendance and achievement

With this system, institutions can take action that suits the needs of each student, based on learning patterns and classroom activeness.
